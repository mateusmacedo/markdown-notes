## Como Criar um Novo Prompt

1. **Seja Específico**: Defina claramente o objetivo do prompt
2. **Contexto**: Forneça contexto necessário
3. **Formato de Saída**: Especifique o formato esperado da resposta
4. **Exemplos**: Inclua exemplos quando possível
5. **Iteração**: Refine baseado nos resultados

## Estrutura de um Prompt

```markdown
# Nome do Prompt

## Objetivo
Descrição clara do que o prompt faz

## Contexto
Informações de background necessárias

## Prompt
```
O texto do prompt aqui
```

## Exemplo de Uso
Como usar e o que esperar

## Variações
Adaptações possíveis do prompt
```
