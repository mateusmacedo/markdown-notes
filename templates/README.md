# Templates

Esta pasta contém templates reutilizáveis para criar documentos em cada categoria.

## Templates Disponíveis

- `adr-template.md` - Architecture Decision Record
- `projeto-template.md` - Documento de Projeto
- `prompt-template.md` - Prompt de IA
- `diagrama-template.md` - Documento de Diagrama

## Como Usar

1. Copie o template apropriado
2. Renomeie o arquivo de acordo com seu conteúdo
3. Preencha as seções
4. Salve na pasta apropriada

## Personalizando Templates

Sinta-se livre para adaptar os templates às suas necessidades específicas.
