# Projetos

Esta pasta contém manifestos e documentação relacionada a aspectos de projetos.

## Conteúdo

Aqui você encontrará:

- **Visões de Produto**: Definições de alto nível do produto e seus objetivos
- **Roadmaps**: Planejamento temporal de funcionalidades e entregas
- **Requisitos**: Especificações de requisitos funcionais e não-funcionais
- **Especificações Técnicas**: Detalhes técnicos de implementação
- **Manifestos de Projeto**: Princípios e valores que guiam o projeto

## Estrutura Sugerida

Organize documentos por projeto:

```
projetos/
├── nome-do-projeto/
│   ├── visao.md
│   ├── roadmap.md
│   ├── requisitos.md
│   └── especificacoes-tecnicas.md
└── README.md
```

## Como Criar um Novo Documento de Projeto

1. Crie uma pasta com o nome do projeto
2. Use templates disponíveis em `../templates/`
3. Adapte conforme necessário
4. Mantenha atualizado conforme o projeto evolui
